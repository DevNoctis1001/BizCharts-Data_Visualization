# Bizcharts Plot

> 基于 G2 4.0 版本


# 新特性
- Tooltip 全面支持React 定制化。
- createPlot 快速使用g2Plot，并通过bizcharts 进行微定制。


## npm 使用

```bash
npm install bizcharts --save
```

## Browser

```html
<script src="https://unpkg.com/bizcharts@${version}/umd/BizCharts.min.js"></script>

```

## 开发

```bash
tnpm i
tnpm run start
```

## 发布

```bash
npm publish
```

在 **stories**中开发对应的调试 demo
